F1 Stats: Uso de APIs File, Canvas y SVG

Esta herramienta permite cargar un archivo de texto con datos de pilotos de F1 y generar un gráfico de barras. El archivo debe tener el formato: 

Nombre,Posición,Carrera
Pilot1,1,Grand Prix 1
Pilot2,2,Grand Prix 2
...

Después de cargar el archivo, el gráfico se mostrará en el canvas, y podrás descargarlo como un archivo SVG haciendo clic en el botón correspondiente.
